🛠️ Project Name

Digital Tools Platform

📄 Description

This is a simple React web app where you can see different digital products and add them to your cart. You can check the product details, see your cart, and proceed to checkout.


⚙️ Technologies Used
1/ React.js
2/ Tailwind CSS & DaisyUI
3/ JavaScript (ES6+)
4/ React-Toastify
5/ JSON (for product data)
🌟 Features

Product Listing – View all products in a neat 3-column layout with details like name, price, features, and icon.

##Cart System – Add products to the cart, see the total count, remove items, and proceed to checkout.

##Responsive Design – Works on mobile, tablet, and desktop.


✨ Optional Features
1/ Buy Now button increases cart count in Navbar.
2/ Toast message shows “Added to cart” when
3/ product is added.
4/ Proceed to checkout clears all items from  cart.



📝 How to Use
1/ Open the project in a browser.
2/ Browse products in the Product section.
3/ Click Buy Now to add a product to the cart.
4/ Click Cart button to see your selected 5/ products.
5/ Remove products using Remove button or click Proceed to Checkout to clear the cart.
